## Reproduction

We recomend you to setup in the following steps:

1. Clone the repo.
```
git clone https://github.com/ZxyLinkstart/DAA-GNN
```

2. Create conda environment
```
conda env create -f environment.yml
conda activate DAA-GNN
```
